Overview

This project is a small quantitative simulator that asks a simple question: “What happens to different groups when certain popular policy ideas are implemented?”

Focusing on proposals associated with Zohran Mamdani, the simulator models two scenarios:

  • Rent freeze – tenants pay the same rent for a set period instead of facing normal yearly increases.
  • Free buses – bus fares are removed, changing costs for passengers, public transport, and taxi drivers.

The goal is not to take a political side, but to practice thinking like a quant: define assumptions, run calculations, and see how the numbers move for each group.

Methodology

The simulator is built as a small, parameter-based Python model:

  • For the rent freeze, it uses assumptions like: number of rent-stabilised units, average monthly rent, normal yearly increase, and freeze duration. From this, it estimates total tenant savings and landlord revenue loss.
  • For free buses, it uses: daily bus trips, average bus fare, daily taxi trips, average taxi fare, and a percentage of taxi users who might switch to free buses. From this, it estimates passenger savings, MTA ticket revenue loss, and taxi driver income loss.

Under the hood, everything is simple arithmetic plus basic sensitivity analysis. You can tweak parameters in the code to explore “what if” scenarios.

How to run the project

Basic steps to run the simulator locally:

  • Clone the repository from GitHub.
  • Install Python 3 and dependencies from requirements.txt.
  • Run main.py to generate the numbers and charts.

Summary:

  • rent_freeze_analysis.py – logic for rent freeze modelling.
  • free_buses_analysis.py – logic for free buses and taxi impact.
  • main.py – prints a summary and saves charts into the images folder.

The code is intentionally kept simple and heavily parameterised so it can be extended later with real-world datasets (e.g., MTA reports, NYC housing data).
